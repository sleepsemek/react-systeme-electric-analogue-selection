# AI Impulse Hackathon — Expert Search (Frontend)

**Expert Search** — веб-приложение для интеллектуального поиска аналогов продукции компании **Systeme Electric**, разработанное в рамках хакатона AI Impulse.
---
Выполнен на:
 - React + Vite
 - TypeScript
 - TailwindCSS
---
Основной функционал:
 - Загрузка номенклатуры в .xslx
 - Парсинг таблицы в JSON, получение таблицы аналогов
 - Отображение результатов, корректировка с обратной связью
 - Экспорт аналогов в .xslx
